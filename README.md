# SplitEase 分帳程式

簡單好用的多人分帳工具，支援即時同步！

## 🚀 快速開始

### 後端
```bash
cd backend
npm install
npm start
```

### 前端
```bash
cd frontend
npm install
npm run dev
```

## 📱 功能

- ✅ 建立專案（聚餐、旅行等）
- ✅ 產生邀請碼分享給朋友
- ✅ 新增支出記錄
- ✅ 自動計算誰欠誰多少
- ✅ 一鍵結清功能
- ✅ 即時同步（每 5 秒自動更新）

## 🔧 技術

- **後端**: Node.js + Express
- **前端**: React + Vite + Tailwind CSS
- **部署**: Vercel (前端) + Railway (後端)
